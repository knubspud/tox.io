Summary: Latitude vs. X

For this study we analyzed a dataset of over 500 (random) cities, each with unique latitudes. On Earth, weather is correlated with latitude. The OpenWeatherMap API provides data sets for each city. This dataset was analyzed and the results are presented as scatter plots.

Various software tools were used. Matplotlib was used to create the plots. Python was used to export HTML, and Bootstrap/CSS/HTML was used to create this site. Please navagate the site and/or click on the various visualizations to see detail. For additional questions, please contact Cheryl Hornung.
